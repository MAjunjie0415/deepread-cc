**新增环境变量**：
```bash

```

### 🚀 使用方法

#### 1. 获取 Supadata API Key
```bash
# 访问 https://supadata.ai
# 注册并获取 API Key
```

#### 2. 配置环境变量
```bash
# 创建 .env.local
echo "SUPADATA_API_KEY=sup_your_api_key" >> .env.local
echo "DEEPSEEK_API_KEY=sk-your_api_key" >> .env.local
```

#### 3. 测试配置
```bash
npm run test:api
# 或
node test-supadata.js
```

#### 4. 启动应用
```bash
npm run dev
```
